# pyprogen

## Description

Generating Python project from Command Line Interface.

```
pyprogen
```

The package will have th following structure with a connection with Github and Gitpage.

```
pyprogen
    ├── pyprogen
    │   ├── __init__.py
    │   ├── __main__.py
    │   ├── __version__.py
    │   ├── ressources
    │   │   ├── __init__.py
    ├── tests
    |   ├── __init__.py
    ├── examples
    ├── laboratory
    ├── venv
    ├── docs
    │   ├── source
    │   │   ├── api_doc
    │   │   ├── usage_doc
    |   |   ├── conf.py
    |   |   ├── index.rst
    |   |   ├── api.rst
    |   |   ├── usage.rst
    |   ├── build
    ├── .gitignore
    ├── README.md
    ├── requirements.txt
    ├── setup.py
    ├── Makefile
    └── LICENSE
```
